Este es un pequeño boceto hecho con [Next.js](https://nextjs.org.)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Contexto

Junto con mis colegas del instituto estamos desarrollando una pagina web tipo red social para explorar lugares de chile (cerros, museos, senderos, parques, etc). Nuestro valor agregado es mediante Check-in/QR registrar recorridos, sumar puntos e insignias, ver perfiles, comentar y administrar contenidos desde un panel administrador.

Como somos novatos en el mundo de la programación, esta idea irá mutando con el tiempo. Lo que se aprecia hoy, será muy disntinto a lo que será en un futuro.
